FAQ

Pertanyaan yang Sering Ditanya

Berapa lama waktu pengerjaan desain UI/UX? +
Tergantung kompleksitas proyek. Landing page sederhana 3–5 hari kerja, aplikasi multi-halaman 2–4 minggu. Kami selalu memberikan estimasi waktu di awal proyek.
Apakah ada revisi yang diperbolehkan? +
Ya! Kami menyediakan revisi unlimited selama dalam lingkup proyek yang telah disepakati. Revisi di luar scope dapat didiskusikan dengan biaya tambahan yang wajar.
Format file apa yang akan saya terima? +
Untuk desain: file Figma, PDF, PNG, SVG. Untuk web: source code HTML/CSS/JS atau file proyek sesuai framework. Semua aset akan diserahkan dengan rapi dan terdokumentasi.
Bagaimana sistem pembayarannya? +
Kami menggunakan sistem 50% di awal sebagai DP, 50% sisanya setelah proyek selesai dan disetujui. Untuk proyek besar, bisa dicicil per milestone.
Apakah bisa konsultasi gratis dulu? +
Tentu! Konsultasi awal 30–60 menit sepenuhnya gratis tanpa komitmen apapun. Gunakan waktu ini untuk mendiskusikan ide dan kebutuhan Anda.
